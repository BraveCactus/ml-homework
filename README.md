# ml-homework

В данном домашнем задании используется пакетный менеджер uv

**установка uv:**  
`pip install uv`

**создание виртуального окружения (Python 3.14):**  
`uv venv --python 3.14`

**активация виртуального окружения:**  
`.venv\Scripts\activate`

**установка зависимостей из pyproject.toml:**  
`uv sync`
